# For developers (self-host)

outsideclaw is a vertical outdoor assistant built on top of OpenClaw.

## Recommended setup
- Use OpenClaw as the Telegram gateway/agent runtime
- Use outsideclaw as the deterministic local toolkit (DB, import, guide, recommend, share)

## One-click integration
See: `docs/openclaw-integration.md`

## Core principles
- Offline-first
- Deterministic-by-default
- Low-token outputs
- Privacy-first
