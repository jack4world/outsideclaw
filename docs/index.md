# outsideclaw

你的户外安全助手（基于 **OpenClaw**）。

- 导入路线（GPX/KML）
- 徒步中发定位 → 偏航纠正 + 关键点提醒 + 天气剧变预警
- 离线优先、隐私优先

## 我是徒步爱好者
想直接用服务（Telegram 交互）：
- 见：`docs/for-hikers.md`

## 我是开发者
想自部署（OpenClaw 接入 + outsideclaw 本地工具链）：
- 见：`docs/for-developers.md`
- 一键接入：`docs/openclaw-integration.md`

## 为什么要用 outsideclaw
- **离线可用**：关键时刻没网也能算偏航/指路
- **确定性优先**：核心计算不依赖 LLM，更稳定
- **低 token 输出**：机器行 + 简短中文，适合 agent 工作流
- **隐私优先**：默认本地存储，分享需显式操作
