# Gear list (2-person, 1-night ridge/summit camp)

## Must
- Shelter: tent + stakes + guylines
- Sleep: sleeping bag (rated for windchill) + sleeping pad
- Light: headlamp per person + spare power
- Power: power bank
- Clothing: wind/rain shell + insulation layer + spare base + hat + gloves
- Safety: basic first-aid + emergency blanket + whistle
- Water: assume no refill; carry enough
- Food: dinner + breakfast + trail snacks + 20% buffer

## Strongly recommended
- Extra guylines, stronger stakes, tape/zip ties
- Electrolytes
- Trekking poles

## Optional
- Knee support, camp light, sit pad
